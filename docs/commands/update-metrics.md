# vsc update-metrics

Update text metrics from Renderer measurement (Phase 10 Q→P bridge)

## Usage

```
vsc update-metrics --id <ID> --width <WIDTH> --height <HEIGHT>
```

## Options

Options:
      --id <ID>          Text entity ID
      --width <WIDTH>    Measured width
      --height <HEIGHT>  Measured height
  -h, --help             Print help
