# vsc add-entity

Add a new entity to the constraint graph (Phase 10)

## Usage

```
vsc add-entity [OPTIONS] --entity-type <ENTITY_TYPE>
```

## Options

Options:
  -t, --entity-type <ENTITY_TYPE>  Entity type (text)
  -c, --content <CONTENT>          Content (for text entities)
      --font-family <FONT_FAMILY>  Font family (for text entities)
      --font-size <FONT_SIZE>      Font size (for text entities)
  -x, --x <X>                      Initial X position
  -y, --y <Y>                      Initial Y position
  -h, --help                       Print help
