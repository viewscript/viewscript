# vsc patch-constraint

Modify an existing constraint on an entity

## Usage

```
vsc patch-constraint [OPTIONS] --entity-id <ENTITY_ID> --component <COMPONENT> --relation <RELATION> --value <VALUE>
```

## Options

Options:
      --entity-id <ENTITY_ID>  Entity ID to modify
      --component <COMPONENT>  Component to modify (x, y, width, height, etc.)
      --relation <RELATION>    New relation (eq, le, ge)
      --value <VALUE>          New value (rational, e.g., "100" or "100/1")
      --intent <INTENT>        Natural language intent
  -h, --help                   Print help
