# vsc build

Build the project for a target renderer

## Usage

```
vsc build [OPTIONS]
```

## Options

Options:
  -t, --target <TARGET>  Target renderer (wgpu, webgl, svg) [default: wgpu]
  -o, --outdir <OUTDIR>  Output directory [default: dist]
  -h, --help             Print help
