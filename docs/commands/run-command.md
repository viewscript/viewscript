# vsc run-command

Run a CODL command file (Phase 15)

## Usage

```
vsc run-command [OPTIONS] --args <ARGS> <COMMAND_FILE>
```

## Options

Options:
      --args <ARGS>      Arguments as JSON object
      --intent <INTENT>  Natural language intent
  -h, --help             Print help
